# Softbodies

Demonstrating the basics of Softbodies.
Article: https://devlog.boredfish.com/2026/04/01/softbody-chaos/

The key steps are organised as:

- Step 0 : Prep: Setting up the canvas
- Step 1 : Basic movement "hello, world"
- Step 2 : Integration - gravity/forces
- Step 3 : Spring Joints
- Step 4 : Collisions
- Step 5 : Optimisation (broad phase)

Just for fun/curiosity, there's even a basic softbody test written in PHP and outputting to the commandline...
